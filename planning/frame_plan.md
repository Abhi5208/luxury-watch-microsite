# FRAME_PLAN — luxury-watch-microsite

## Goal

Define exactly which visuals need to be generated before starting asset creation.

---

## Priority

Generate frames for Home only.

Pages:

- Home → full visual priority
- About → static/supporting visuals later
- Contact → no special asset generation needed

---

## Home Sections Requiring Visual Planning

### 1. Hero Reveal

Purpose:

- premium first impression
- watch introduced in darkness

Needed frames:

- Start: near-black background, minimal visible silhouette
- Mid: partial reveal with dramatic lighting
- End: clear hero watch shot

Asset type:

- hero stills or short layered sequence

---

### 2. Macro Details

Purpose:

- show materials and luxury finish

Needed frames:

- case close-up
- dial detail
- crown / edge / glass highlight
- bracelet or strap texture

Asset type:

- separate stills or short transitions

---

### 3. Dissection Sequence

Purpose:

- main wow moment
- watch separates into components

Needed frames:

- Start: fully assembled watch
- Key state 1: outer layers separating
- Key state 2: more internal structure visible
- Key state 3: exploded view with distinct layers
- End: fully dissected composition

Asset type:

- frame sequence
- highest generation priority

---

### 4. Movement Showcase

Purpose:

- emphasize precision and mechanics

Needed frames:

- internal movement hero
- gear/mechanism close-up
- depth/mechanical layering shot

Asset type:

- stills or short motion sequence

---

### 5. Reassembly

Purpose:

- visual closure
- parts return into final product

Needed frames:

- Start: dissected view
- Mid: components converging
- End: nearly assembled
- Final: complete watch restored

Asset type:

- frame sequence, possibly derived from dissection logic

---

### 6. Final Showcase + CTA

Purpose:

- polished final impression

Needed frames:

- fully assembled beauty shot
- alternate angle if needed
- CTA-supporting background visual

Asset type:

- high-quality still or subtle motion background

---

## Frame Generation Strategy

### Use Whisk for:

- key start frames
- key end frames
- standout composition frames
- style consistency exploration

### Use Flow for:

- interpolation between selected key frames
- motion exploration for dissection/reassembly
- generating filler sequence candidates

---

## Generation Priority Order

1. Dissection start + end
2. Hero reveal start + end
3. Reassembly start + end
4. Macro detail stills
5. Movement showcase stills
6. Final CTA visual

---

## Open Decisions

- exact visual style reference
- watch design language
- number of total dissection layers
- dark luxury vs brighter editorial lighting
- frame count for sequence sections
- whether reassembly is reverse of dissection or separately generated
