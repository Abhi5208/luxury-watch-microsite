# Agent Roles

## CTO

- Owns technical direction.
- Resolves tradeoffs.
- Approves architecture changes.
- Protects delivery scope.

## Architect

- Defines system structure.
- Maintains ADRs.
- Reviews API and data models.
- Identifies technical risks.

## Builder

- Implements approved tasks.
- Keeps code inside `/code`.
- Updates build notes.
- Flags blockers early.

## QA

- Owns test planning.
- Records findings.
- Runs regression checks.
- Verifies release readiness.

## Reviewer

- Reviews implementation quality.
- Checks scope alignment.
- Confirms acceptance criteria.
- Identifies missing tests.
