# SOW — luxury-watch-microsite

## Project Overview

Design and develop a small premium brand site for a luxury watch concept, with Home as the high-end scroll-driven storytelling experience.

This project is a portfolio-focused build intended to demonstrate:

- advanced UI/UX
- animation capability
- performance awareness
- clean engineering practices

---

## Scope of Work

### Included

- Multi-page brand site with Home, About, and Contact
- Home scroll-based storytelling experience
- Visual sections:
  - Hero reveal
  - Macro details
  - Dissection sequence
  - Movement showcase
  - Reassembly
  - Final CTA
- Lightweight About page focused on brand philosophy and craft
- Contact page with UI-only inquiry form, client-side validation, and toast feedback on submit
- Frame-based animation integration
- Responsive layout (desktop-first)
- SEO baseline implementation
- Accessibility baseline implementation
- Performance optimization (Lighthouse targets)

---

### Not Included

- Full e-commerce system
- Shop / product purchasing flow
- Product configurator
- Blog
- CMS integration
- Auth
- Backend services
- API submission
- Email sending
- Real contact handling

---

## Deliverables

- Functional deployed premium brand site
- Source code repository
- Optimized visual assets
- Lighthouse report (performance, SEO, accessibility)
- Portfolio-ready demo (screenshots / video)

---

## Assumptions

- Visual assets will be AI-generated (Whisk + Flow pipeline)
- Desktop experience is primary
- Mobile experience may be simplified
- Contact form is UI-only and does not send data externally
- Any displayed contact email must be fake placeholder copy, e.g. `hello@brand.com`
- No API integrations are required

---

## Constraints

- Must meet defined Lighthouse targets
- Must maintain smooth Home scroll experience on modern desktop browsers
- Must avoid heavy blocking scripts
- Must remain deployable on standard platforms (e.g., Vercel/Netlify)
